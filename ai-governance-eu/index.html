Talon 2.0 thesis

Evidence-grade governance is enforced before execution.

Talon’s wedge is narrow by design: it is a self-hostable network chokepoint that turns AI traffic into signed, regulator-mapped evidence. It does not try to be the broadest AI gateway or the deepest in-process agent framework.

🔐

PII and content controls

Detect, redact, warn, or block sensitive data before it reaches an LLM provider, using EU-relevant recognizers such as IBAN, VAT IDs, and national IDs.

🇪🇺

Sovereignty routing

Use EU strict, EU preferred, or global modes to decide which providers and regions are acceptable for a given data tier.

🧾

Signed evidence

Record caller, tenant, policy decision, PII findings, model, cost, hashes, and signature so auditors receive artifacts, not screenshots.

Buyer positioning

Use Talon when the buyer needs defensible evidence.

The best-fit buyer is an EU or EMEA team that already has LLM or agent traffic and now needs to pass a customer, DPO, security, board, or auditor review without sending governance data to another managed control plane.
